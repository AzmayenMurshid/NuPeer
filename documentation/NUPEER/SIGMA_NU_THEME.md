# Sigma Nu Theme Implementation

## Color Scheme

The UI has been updated to align with **Sigma Nu fraternity** official colors with a modern 3D shadow design:

### Background Colors
- **Light Mode**: Dark gold background (#d97706) - Rich Sigma Nu gold
- **Bright Setting**: Light gold-tinted background (#fef3c7) - Soft gold tint
- **Dark Mode**: Matt grey-black background (#1a1a1a) - Deep matte finish

### Primary Colors (Gold/Yellow)
- **Gold**: #f59e0b - Sigma Nu's signature gold for buttons and accents
- **Dark Gold**: #d97706 - Used for light mode background
- **Light Gold**: #fef3c7 - Used for bright setting background
- **Gradients**: Various shades from light gold to dark gold

### Design Style
- **No Borders**: Clean borderless design
- **3D Shadows**: Deep shadows (shadow-xl, shadow-2xl) for depth and texture
- **Hover Effects**: Enhanced shadows on hover for interactive feedback

## Design Elements

### Sigma Nu Branding
- **Greek Letters (ΣΝ)**: Added to headers and key sections
- **Gold Borders**: All cards and containers use gold-tinted borders
- **Gold Buttons**: Primary actions use Sigma Nu gold
- **Gold Hover Effects**: Interactive elements highlight in gold

### Light Mode
- White/light gray backgrounds
- Black/dark gray text
- Gold (#f59e0b) for primary actions and accents
- Light gold borders (#fef3c7) for subtle definition

### Dark Mode
- Dark gray/black backgrounds
- Light gray/white text
- Lighter gold tones for better contrast
- Darker gold borders for definition
